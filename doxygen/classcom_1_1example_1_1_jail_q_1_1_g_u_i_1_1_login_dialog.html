Detailed Description

Diálogo modal de inicio de sesión del sistema JailQ.

Se abre sobre JailQMainGUI cuando el usuario pulsa «Iniciar sesión». Al cerrarse, la ventana principal consulta isLoginCorrecto(), getUsername() y getTipoCuenta() para actualizar su estado de sesión.

Envía las credenciales al endpoint POST /cuentas/login/policia y extrae el tipoCuenta del JSON de respuesta sin depender de ninguna librería JSON externa.
