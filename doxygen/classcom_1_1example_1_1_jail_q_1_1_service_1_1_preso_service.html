Detailed Description

Servicio para gestionar operaciones sobre Preso.

  • Esta clase se encarga de validar y guardar objetos Preso en la base de datos utilizando PresoDAO. Incluye validaciones de nombre, apellidos, edad, condena y fechas de ingreso.
  • Anotada con Service para ser detectada como componente de Spring.

◆ anadirPreso()

Preso anadirPreso ( Preso nuevoPreso)

Añade un nuevo Preso a la base de datos tras realizar validaciones.

  • Validaciones realizadas:
    • Compronar que existe el objeto
    • Validación de Nombre y Apellidos (con trim para detectar "" también como vacío)
    • Validacion de edad
    • Comprobacion de condena.
    • Validacion de fecha de ingreso -> no puede ser futura y tampoco puede entrar antes de haber nacido
  • Mensajes de error se imprimen en consola en caso de datos inválidos. Si todas las validaciones pasan, el preso se guarda mediante PresoDAO#save(Object).
  • Parameters
    nuevoPresoEl objeto Preso a añadir. No puede ser null.
