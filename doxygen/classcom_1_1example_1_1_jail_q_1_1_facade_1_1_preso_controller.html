Detailed Description

Controlador REST encargado de gestionar las operaciones relacionadas con los presos.

  • Expone endpoints HTTP para:
    • Crear presos
    • Obtener todos los presos
    • Obtener un preso por ID
    • Eliminar un preso
  • Base URL: /preso
