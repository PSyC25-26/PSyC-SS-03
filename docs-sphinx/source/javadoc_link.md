# Doxygen API Documentation

Full technical documentation is generated by **Doxygen** from the JavaDoc
comments in the source code and published as part of the GitHub Pages deployment.

## Accessing the Documentation

**[-> Open Doxygen Documentation](../doxygen/index.html)**

## Generating Doxygen Locally

```bash
# From JailQ/ folder
mkdir -p target/doxygen
doxygen src/main/resources/Doxyfile

# Open result (Windows)
start target/doxygen/html/index.html
```

## Doxygen vs Javadoc

| Tool | Output | Strength |
|------|--------|----------|
| **Doxygen** | `doxygen/html/` | Collaboration diagrams, cross-references, rich navigation |
| **Javadoc** | IDE integration | Standard Java API reference |

Doxygen is fully compatible with the JavaDoc format (`/** ... */`), so all
existing comments in the code are used directly.
